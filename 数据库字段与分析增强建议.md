# 数据库字段与分析增强建议

## 一、当前数据库数据结构

当前主数据来自 `全行业银行综合分析2020-2025（含PB） (2).xlsx` 的 `综合底表`，共 342 条记录、257 个字段，覆盖 57 家银行、2020-2025 年面板数据。

| 数据模块 | 字段数量 | 可支持的经营分析 |
| --- | ---: | --- |
| 基本信息 | 3 | 银行简称、银行类型、年份，支持目标银行、对标银行、类型均值筛选 |
| 盈利概览 | 11 | 营业收入、净利息收入、手续费净收入、净利润、PPOP、核心营收及增速 |
| 盈利指标 | 8 | ROE、ROA、NIM、净利差、成本收入比、非息占比、真实核心非息、高波动收入 |
| 资产负债 | 8 | 资产、负债、权益、贷款、存款、生息资产、计息负债、存贷比 |
| 息差分析 | 5 | 生息资产收益率、计息负债成本率、NIM缺口、真实存贷利差、核心存款定期化 |
| 资产质量 | 9 | 不良率、拨备覆盖率、贷款拨备率、逾期率、关注率、偏离度、隐性不良暴露 |
| 资本充足率 | 11 | 资本充足率、一级资本、核心一级、资本杠杆率、资本余量、估算RWA、RWA密度 |
| 流动性指标 | 6 | 流动性比率、流动性覆盖率、备付金率、存贷比、单一最大贷款、最大十家贷款 |
| 存款结构 | 8 | 公司/个人存款、定期、活期、保证金存款 |
| 贷款结构 | 7 | 公司贷款、个人贷款、票据贴现及分项不良 |
| 五级分类 | 17 | 正常、关注、次级、可疑、损失、不良合计、贷款损失准备、各档拨备率和占比 |
| 逾期贷款 | 25 | 逾期总额、期限结构，以及信用、保证、抵押、质押逾期结构 |
| 个人贷款分产品 | 11 | 住房、消费、经营、信用卡、其他个贷占比及分项不良 |
| 公司贷款分行业 | 40 | 20 个行业贷款占比与 20 个行业不良率 |
| 金融投资分布 | 12 | 债券、政府债、金融债、企业债、同业存单、基金、信托及理财等 |
| 金融资产三阶段 | 36 | 贷款、金融投资、合计口径的一二三阶段余额、占比、准备、拨备率 |
| 利润表详情 | 13 | 利息收入、利息支出、手续费收入/支出、管理费用、营业利润、所得税等 |
| 金融资产与同业往来 | 11 | FVTPL、FVOCI、AC金融资产、同业资产负债、现金及准备金 |
| 权益构成 | 10 | 股本、资本公积、盈余公积、未分配利润、一般风险准备等 |
| 现金流量 | 3 | 经营、投资、筹资活动现金流净额 |
| 每股与估值 | 3 | 基本EPS、年末PB、年中PB |

## 二、还能进一步做出的高价值分析

| 分析主题 | 可以新增的指标/图表 | 经营含义 |
| --- | --- | --- |
| 盈利真实性 | 真实核心非息、高波动收入、经营现金流/净利润、PPOP与净利润缺口 | 区分主业修复、投资补位、拨备释放和现金流支撑 |
| 息差防守 | NIM缺口、真实存贷利差、核心存款定期化、负债成本 | 识别资产让价后，负债端是否有能力对冲 |
| 存款经营 | 公司/个人活期占比、定期占比、保证金存款 | 判断低成本负债沉淀能力和客群经营基础 |
| 零售风险 | 住房、消费、经营、信用卡占比与不良率 | 拆开“个贷不良”来源，定位按揭、消费信用和小微经营风险 |
| 对公行业风险 | 行业贷款集中度、行业不良率、行业风险贡献 | 找到风险从哪个行业暴露，而不是只看全行不良率 |
| 逾期迁徙 | 逾期账龄结构、担保方式逾期结构、偏离度 | 判断风险是否在结果指标之前已经恶化 |
| 五级分类迁徙 | 关注类占比、次级/可疑/损失占比、各档拨备率 | 判断分类严格度、风险确认速度和拨备充分性 |
| 投资资产波动 | 基金、信托理财、FVTPL、三阶段投资资产 | 判断非息收入是否带有市场波动属性 |
| 三阶段信用风险 | 贷款/投资/合计二阶段和三阶段占比、准备率 | 用预期信用损失口径提前识别风险迁徙 |
| 资本消耗 | RWA增速、RWA密度、资本余量、ROA/PB联动 | 判断扩表是否创造价值，还是消耗资本自由度 |
| 流动性约束 | LCR、流动性比率、备付金率、最大贷款集中度 | 判断增长是否受流动性和集中度约束 |
| 估值解释 | PB、EPS、ROA、风险确认、资本消耗交叉 | 区分“价值错配”和“质量折价” |

## 三、已新增到应用的数据派生字段

| 派生字段 | 计算逻辑 | 用途 |
| --- | --- | --- |
| 利息净收入占比 | 净利息收入 / 营业收入 | 判断收入对利差主业的依赖程度 |
| 手续费收入占比 | 手续费净收入 / 营业收入 | 判断非息收入中稳定服务收入厚度 |
| 核心营收占比 | 核心营收 / 营业收入 | 判断营收是否由主业支撑 |
| 贷款/资产 | 贷款总额 / 资产总计 | 判断资产投放结构和信贷依赖 |
| 存款/负债 | 存款总额 / 负债合计 | 判断负债来源是否稳定 |
| 管理费用/资产 | 管理费用 / 资产总计 | 判断成本刚性是否侵蚀资产回报 |
| 经营现金流/净利润 | 经营活动现金流净额 / 净利润 | 判断利润是否有现金流支撑 |
| 债券/基金/信托理财占资产比 | 各投资资产 / 资产总计 | 判断投资资产对收益波动的贡献 |
| 公司/个人存款占比 | 公司或个人存款 / 存款总额 | 拆分负债客群结构 |
| 公司/个人活期存款占比 | 公司或个人活期 / 存款总额 | 判断低成本资金沉淀能力 |
| 零售最高分项不良率 | 住房、消费、经营贷款不良率最大值 | 定位零售风险最薄弱产品线 |
| 零售分项风险分化 | 三类零售贷款不良率最大值 - 最小值 | 判断个贷风险是否集中在单一产品 |
| RWA增速-净利润增速缺口 | 估算RWA增速 - 净利润增速 | 判断资本消耗是否快于利润修复 |
| RWA增速-资产增速缺口 | 估算RWA增速 - 总资产增速 | 判断扩表是否变得更重资本 |

## 四、产品侧建议

1. 总览页保留少量核心指标：ROA、核心营收、净息差、隐性不良、资本余量、PB。
2. 每个章节下面做专题展开：盈利真实性、负债结构、零售风险、对公行业、投资资产、三阶段风险、资本估值。
3. 每张图的讲解应由当前选择的数据生成，至少包含目标银行、对标组、类型均值、分位、差距和建议动作。
4. 数据覆盖章节应作为正式报告附录，披露指标完整性，避免用缺失较多的指标生成强结论。
5. 导出数据应保留多工作表：选择摘要、目标银行、对标银行、类型均值、完整性、全样本明细。
