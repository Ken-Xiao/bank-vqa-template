# Sprint 2 选择与总览改造记录

更新时间：2026-05-31

## 已完成

1. 目标银行选择深钻
   - 目标银行区增加当前银行摘要卡，展示银行类型、区域、ROA、净息差、PB。
   - 保留“大行 / 股份行 / 城商行 / 农商行”类型筛选，城农商场景继续支持区域筛选。
   - 目标银行列表高度收敛，确认前页面不再被长列表撑开。

2. 对标银行选择降噪
   - 对标区增加组合摘要卡，展示当前对标模板、已选银行、ROA 均值和 PB 均值。
   - 对标候选按“已选 / 推荐 / 其他”排序，减少从全量银行列表中寻找推荐样本的成本。
   - 推荐样本增加视觉标识，便于快速校验对标组是否合理。

3. 总览首屏重排
   - 将“董事会议题”和“SPARC 交通灯”前移到 30 秒结论任务台。
   - 将“推荐下钻路径”和“What-if 决策推演”下沉为第二层工具区。
   - 移除原先在价值创造区和行长摘要区的重复首屏占位，保持动态渲染 id 唯一。

4. 总览深层模块折叠
   - 新增“总览密度”开关，默认收起价值创造、SPARC 全景和投行证据等深层模块。
   - 深层模块通过 `data-overview-depth="secondary"` 标记，便于后续继续纳入折叠体系。
   - 用户需要完整分析时可一键展开，默认总览更接近董事会入口而不是功能堆叠页。
