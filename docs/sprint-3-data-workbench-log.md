# Sprint 3 数据可信度工作台记录

更新时间：2026-05-31

## 已完成 V1

1. Metric Explorer 指标探索器
   - 数据页新增“指标探索器”子视图。
   - 支持按主题选择任一指标，展示目标银行当前值、对标组均值、类型均值、全样本分位和报告使用建议。
   - 单指标页同步展示近年趋势、来源字段、公式、业务含义和口径风险说明。

2. 口径风险 L1-L4
   - 新增 `metricCalibrationRisk()`，统一输出 L1/L2/L3/L4、颜色、报告用途和说明。
   - L1：主报告；L2：主报告+脚注；L3：附录；L4：待补。
   - 原“口径统一与可比性”卡片已接入同一套 L1-L4 逻辑。
   - 专题事实包和结构化事实包已写入口径风险等级、报告使用建议和口径脚注。
   - AI/模板叙事已按 L3/L4 自动降低强结论表达。
   - 正式报告专题页证据表新增“口径风险”列，并对 L2/L3/L4 指标输出脚注。
   - PPTX 导出页脚已接入口径风险提示；无风险脚注时回落到默认数据来源行。

3. 字段可用性热力图
   - 数据页新增按主题聚合的可用性热力图。
   - 每个主题展示平均覆盖率、指标数量和 L1-L4 等级，用于快速判断哪些专题可以进入主报告。
   - 已接入 `data_governance/field_coverage_matrix.json`，优先按 257 个原始字段的 source_group 输出字段级覆盖热力图。
   - 字段级热力图展示已接入/总字段数、待接入字段示例和优先级排序。

4. 指标手册 V1 联动
   - 复用现有指标字典、公式、来源字段、业务含义和缺失策略。
   - Explorer 与指标详情弹窗共享 `metric_dictionary.json` 和 `analysis_rules.json`。

5. 导出元数据与交付留痕
   - 数据底稿新增“导出元数据”表，记录目标银行、对标组、年份、报告版本、规则版本、导出范围和 L1-L4 口径风险分布。
   - 数据底稿新增“口径风险元数据”表，逐指标输出口径风险等级、报告使用建议、口径脚注、来源字段、公式和覆盖银行。
   - 数据底稿新增“字段覆盖矩阵”表，保留 257 个原始字段的接入状态、优先级和处理建议。
   - “指标完整性”“指标口径”“图表事实包”同步增加 L1-L4 风险字段，方便交付复核。

6. 257 字段展开筛选表
   - 数据页新增字段矩阵展开表，可按全部、待接入、已接入、中优先级、暂缓字段筛选。
   - 展开表保留字段组、字段、状态、优先级和处理建议，方便判断零售风险、行业集中度、迁徙矩阵等专题是否具备底层字段基础。
   - 展示层限制首屏最多 80 行，避免数据页被 257 字段直接撑开。
